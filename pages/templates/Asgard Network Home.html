Welcome to the Asgard network!

What is this?

The Asgard network is an experimental  network currently made up of virtual machines designed to be a modern take on the old look and feel of the classic internet.
Currently it is in the prototyping phase and has several games located on the "Odin" host, along with a classic BBS and a few other things waiting to be discovered.


What hosts are available?

Odin
 Main server host, can be accessed via ssh or telnet and has many games and services
Tyr
 Network router
Freya
 BBS and mail server (The server hosting this site)
Sol
 An emulated Sparcstation 5 running Solaris 7, has a dedicated webpage for access.

Current Plans:

Right now we are planning to make this a greater overlay net over the internet to connect to other universities and create a new "Retro Internet" with modern technology.
Still, Asgard.net needs to be polished some.
